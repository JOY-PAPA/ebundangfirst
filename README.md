# e편한세상 분당 퍼스트빌리지 옵션 계산기

새로 업로드한 엑셀 파일의 `옵션DB` 시트를 기준으로 만든 정적 웹 계산기입니다.

## 구성

- `index.html`: 화면 구조
- `styles.css`: 반응형 화면 스타일
- `app.js`: 선택, 합계, 필터, 공유, CSV, 인쇄 로직
- `option-data.js`: 엑셀 `옵션DB`에서 추출한 옵션 데이터
- `assets/bg_visual_pc.png`: 헤더 배경 디자인 이미지
- `.nojekyll`: GitHub Pages 정적 배포용 파일

## 로컬 확인

```bash
py -m http.server 8100
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8100
```

## 데이터 수정

엑셀 `옵션DB` 시트의 열 구조는 아래 순서를 기준으로 합니다.

```text
ID, 구분, 품목, 구성/조건, 제조사, 택1그룹, 51A, 55A, 55B, 59A, 59T, 비고
```

가격 빈칸은 웹에서 `해당 없음`으로 표시되며 선택할 수 없습니다.

## 주의

본 계산기는 입주예정자 편의용 참고 계산기입니다. 실제 계약 전에는 공식 공고문, 계약서, 견본주택 안내를 최종 기준으로 확인해야 합니다.
