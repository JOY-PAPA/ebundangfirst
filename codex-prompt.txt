아래 저장소는 e편한세상 분당 퍼스트빌리지 입주예정자를 위한 옵션 선택 계산기입니다.

목표:
- 정적 웹사이트 형태의 옵션 계산기를 GitHub Pages로 배포할 수 있게 유지·개선해 주세요.
- 주택형별 옵션 금액, 선택 체크, 총액 합산, 구분별 합계, 선택내역 복사, 공유 링크, CSV 다운로드, 인쇄/PDF 저장 기능이 안정적으로 작동해야 합니다.

중요 요구사항:
1. 서버, DB, 로그인, 개인정보 입력 기능은 추가하지 마세요.
2. 옵션 데이터는 option-data.js에 유지하고, 계산 로직은 app.js에 유지하세요.
3. exclusiveGroup 값이 같은 옵션은 택1입니다. 하나를 선택하면 같은 그룹의 기존 선택이 자동 해제되어야 합니다.
4. 가격이 null인 주택형 옵션은 선택 불가 처리해야 합니다.
5. 조건부 옵션은 경고/점검사항으로 표시하되, 공식 계약 전 확인이 필요하다는 문구를 유지하세요.
6. 모바일 카카오톡 인앱브라우저에서도 보기 편해야 합니다.
7. GitHub Pages에서 바로 동작해야 하므로 불필요한 빌드 도구나 외부 라이브러리는 추가하지 마세요.
8. README.md에 배포 및 데이터 수정 방법을 유지·보완해 주세요.

작업 후 확인:
- index.html을 열었을 때 오류 없이 화면이 나오는지 확인
- 각 주택형별 금액 표시 확인
- 옵션 선택 시 총액, 구분별 합계, 선택 내역이 즉시 바뀌는지 확인
- 택1 그룹 자동 해제 확인
- 공유 링크 복사 후 새 창에서 선택값 복원 확인
- CSV 다운로드 확인
